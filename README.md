# **Exploratory Data Analysis of Hotels in Booking.com**

## Overview

This repository contains an exploratory analysis of hotels listed on Booking.com. The analysis aims to provide insights into various aspects of the hotels, such as pricing, customer ratings, and location distribution.

## Dataset
The data in this example is originally from the article Hotel Booking Demand Datasets (https://www.sciencedirect.com/science/article/pii/S2352340918315191), written by Nuno Antonio, Ana Almeida, and Luis Nunes for Data in Brief, Volume 22, February 2019.

The data was downloaded and cleaned by Thomas Mock and Antoine Bichat for #TidyTuesday during the week of February 11th, 2020 (https://github.com/rfordatascience/tidytuesday/blob/master/data/2020/2020-02-11/readme.md).
